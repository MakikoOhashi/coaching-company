# Vision

## Project Thesis

This project aims to build an AI-run certification coaching company.

The core value is not motivation. The core value is operational relief.

The system should remove the student's planning burden by continuously managing:

- roadmap updates
- daily task selection
- progress tracking
- risk detection
- escalation when judgment becomes uncertain

## What Problem It Solves

Many learners do not fail because they lack materials. They fail because they carry too much management overhead:

- deciding what to study today
- recovering from missed days
- estimating whether they are on track
- adjusting the plan after weak performance

This project treats those tasks as company operations that can be delegated to AI.

## Product Direction

The product should feel less like a chatbot and more like a coaching company that runs in the background.

The user should be able to rely on the system for:

- today's priorities
- progress visibility
- pass-likelihood updates
- escalation of unusual situations

In this model, the student should not need to build the plan alone.

The student's role is to study and report honestly.

The product's role is to handle:

- planning
- strategy
- coaching direction
- milestone management

The company should also accumulate operational knowledge over time.

The system should not only remember the learner's current state. It should also build internal knowledge about what kinds of interventions and strategies actually work.

## Human Role

Human intervention should be minimized and concentrated at the top level.

Humans should interact primarily with the AI CEO, not with every internal agent.

## Initial Target

The first target can be a concrete qualification roadmap, such as the Japanese property management-related exam the user wants to support.

The implementation should remain flexible enough to later support other exams.

## Initial Concrete Use Case

The first real use case can be a coaching experience for the user's husband preparing for the Japanese property management qualification exam.

This is a strong starting point because:

- the exam scope is structured enough to roadmap
- the learner is real, not fictional
- the product can be demonstrated in Japanese study context
- the interface can still be presented in English for hackathon judges

The product framing can therefore be:

- internally: a roadmap and coaching system for a Japanese qualification
- externally: an English-presented AI coaching company for certification progress management
